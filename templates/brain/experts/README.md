# experts — 蒸馏出的专家(可移植副本)

由 `翼德 蒸馏` 写入。每个专家一份 `<name>.md`,是某人/某领域"怎么写"的近似(原则+置信度+来源声明)。

- 本目录是**可移植副本**(随大脑同步);可召唤的本体是 `~/.claude/agents/<name>.md`(SessionStart 会自动从这里同步过去)。
- `index.md`:名字 | 领域 | 何时用 | 来源 | 置信度。
- 召唤:在任意项目里说 `use <name>`;或 `翼德 专家` 列出/推荐/会诊。

⚠️ 专家是近似,不是本人;正确性以代码库/测试为准。默认用代号、不用同事真名;只蒸馏可迁移模式,不照搬专有源码。
