# distilled — 蒸馏知识库

由 `/yide 蒸馏` 写入,把公司多人代码蒸馏成你**私人、可移植**的知识。

- `architecture/` 架构图(C4/Mermaid)+ "怎么运作"
- `patterns/` 去耦后的可复用模式(再表达,非照抄)
- `decisions/` 设计取舍
- `_raw/` 原始片段(**高风险**:含雇主专有代码,仅逐条确认后才存)
- `index.md` 索引(按 来源项目 / 主题 / ipScope)

⚠️ IP 提醒:默认只蒸馏可迁移的知识与模式;逐字搬运专有源码/密钥有法律风险。云同步盘上尤其当心 `_raw/`。非法律意见,请参照你的雇佣/IP 协议。
