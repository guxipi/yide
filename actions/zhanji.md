# 翼德 · 战绩面板(zhanji)

显示勾哥在 extraction 个人项目里的战绩(纯查看,零打扰)。

## 做什么
1. Bash 运行 `node "${CLAUDE_SKILL_DIR}/scripts/progress.js" stats`,拿到:今日完成 / 累计 / 当前连斩 / 最高连斩 / 打卡链 / 放歌次数 / 已解锁三国称号。
2. 原样念给勾哥,末尾可加一句应景的张飞话(如战绩好:"哇呀呀,好一员猛将!";刚起步:"莫急,来日方长,且看俺与你并肩斩将!")。

## 原则
- 只读不写、不催不逼;打卡链断了也只鼓励不羞辱(对个人项目要轻松)。
- 这是彩蛋,严肃工作时别主动弹。
