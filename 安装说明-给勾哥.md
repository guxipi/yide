# 翼德安装说明(给勾哥)🗡️

翼德是咕鸡给你做的专属秘书:记住你的习惯、帮你把关 Unity、记过的错不再犯。
**全程不用碰系统命令行,跟着点选 / 回答就行。**

> 前提:你电脑装了 **Claude Code**,并且装了 **Node**(Claude Code 一般已带;不确定就问咕鸡)。

---

## 安装(从咕鸡发你的压缩包,不需要 GitHub 账号)

**第 1 步** 咕鸡会发你一个 `yide-plugin.zip`。**解压**它,会得到一个 `yide` 文件夹,放到你好找的地方(比如"文档")。
　［截图位:解压后的 yide 文件夹］

**第 2 步** 复制这个 `yide` 文件夹的路径。
　Windows:在文件夹上 **Shift + 右键 → 复制为路径**。把里面的反斜杠 `\` 换成正斜杠 `/`(例:`C:/Users/你的名字/Documents/yide`)。
　［截图位:复制为路径］

**第 3 步** 打开 Claude Code,在**对话框**里输入这两行(注意:是 Claude Code 的对话框,不是 Windows 的 CMD),各自回车:
```
/plugin marketplace add 这里粘贴你的yide文件夹路径
```
```
/plugin install yide@guji-tools
```
　看到安装成功后,**重启一下 Claude Code**(关掉再打开)。
　［截图位:/plugin install 成功提示］

> 以后咕鸡给你新版本,只要用新的 `yide` 文件夹替换旧的,在对话框输入 `/plugin marketplace update guji-tools` 即可。

---

## 第一次见面(翼德会主动打招呼)

重开后,翼德会自我介绍。跟着他走就行:

1. 他会问你 **几道选择题**(Unity 版本、代码/沟通风格、红线)——**点选即可**。
2. 他会问你 **“大脑放哪”**——
   - 选 **Google Drive 文件夹**(推荐)👉 以后换电脑免重配;
   - 或选 **本地**(只在这台电脑)。
   - 你只需点一下,**不用敲任何命令**。
　［截图位:翼德的选择题界面］

> 如果重开后没看到翼德打招呼,在对话框输入 `/yide:yide-onboard` 即可。

---

## 装好之后(你什么都不用记)

以后正常用 Claude Code 写代码,翼德会自动:
- **开场 brief** 你的偏好和红线,新对话不用重新解释;
- 写 C# 时**按 Unity 手游 best practice 把关**(热路径 GC、过时 API、序列化、资源卫生),只提醒、不乱改;
- 记住你**纠正过的错**,以后任何对话都不再犯。

随时可以喊他:`/翼德` 或 `/yide`。

---

## 常见问题(Yes/No 就能解决)

- **想换电脑?** 把你选的那个 Google Drive 文件夹同步过去,装好翼德跑一次 onboard,他会“认出你”,不用重配。
- **不想自动放歌?** 直接跟翼德说“静音奖励”。
- **整理记忆要我管吗?** 不用,翼德每天第一次开工自动整理。
- **卡住了?** 直接在对话框问翼德,或找咕鸡。
