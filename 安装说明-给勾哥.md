# 翼德安装说明(给勾哥)🗡️

翼德是咕鸡给你做的专属秘书:记住你的习惯、帮你把关 Unity、记过的错不再犯。
**全程不用碰系统命令行,跟着点选 / 回答就行。**

> 前提:你电脑装了 **Claude Code**,并且装了 **Node**(Claude Code 一般已带;不确定就问咕鸡)。

---

## 安装(2 步,在 Claude Code 对话框里输入,不是 Windows 的 CMD)

**第 1 步** 打开 Claude Code,在对话框输入这一行,回车 —— 添加咕鸡的插件源:
```
/plugin marketplace add guxipi/yide
```
　看到 “added / 已添加” 就对了。
　［截图位:输入 /plugin marketplace add 后的结果］

**第 2 步** 再输入这一行,回车 —— 安装翼德:
```
/plugin install yide@guji-tools
```
　看到安装成功后,**重启一下 Claude Code**(关掉再打开)。
　［截图位:/plugin install 成功提示］

**第 3 步(强烈建议)开启自动更新** —— 这样以后咕鸡推新版,你**啥都不用管,开机自动拉最新**:
　输入 `/plugin` → 切到 **Marketplaces** 标签 → 选 **guji-tools** → 把 **Auto-update 打开**。
　(第三方插件源默认是**关**的,所以必须手动开一次,否则永远不会自动更新。)

> **关于更新:**
> - **开了自动更新**(第 3 步)→ 以后开机自动拉最新;更新了会提示你敲一下 **`/reload-plugins`**(热加载,**不用重启**)。
> - **没开 / 想手动**:输入 **`/plugin marketplace update guji-tools`** 再 **`/reload-plugins`**。
> - 不管哪种,**你攒下的记忆/教训都不会丢**。

---

## 第一次见面(翼德会主动打招呼)

重开后,翼德会自我介绍。跟着他走就行:

1. 他会问你 **几道选择题**(Unity 版本、代码/沟通风格、红线)——**点选即可**。
2. 他会问你 **“大脑放哪”**——
   - 选 **Google Drive 文件夹**(推荐)👉 以后换电脑免重配;
   - 或选 **本地**(只在这台电脑)。
   - 你只需点一下,**不用敲任何命令**。
　［截图位:翼德的选择题界面］

> 如果重开后没看到翼德打招呼,直接在对话框说一句 **"翼德,磨合一下"** 即可(打命令则是 `/yide:yide onboard`)。
>
> **怎么喊翼德**:它是插件,做不到裸 `/yide`。**最省事是直接用中文喊** —— "翼德 记一下…""翼德 整理""翼德 测一下""翼德 简报";要打命令就是 `/yide:yide <动作>`(带命名空间冒号)。

---

## 装好之后(你什么都不用记)

以后正常用 Claude Code 写代码,翼德会自动:
- **开场 brief** 你的偏好和红线,新对话不用重新解释;
- 写 C# 时**按 Unity 手游 best practice 把关**(热路径 GC、过时 API、序列化、资源卫生),只提醒、不乱改;
- 记住你**纠正过的错**,以后任何对话都不再犯。

随时可以喊他:直接说 **"翼德 …"**(中文最省事);打命令则是 `/yide:yide <动作>`(插件命令带命名空间,**没有裸 `/yide`**)。

---

## 常见问题(Yes/No 就能解决)

- **想换电脑?** 把你选的那个 Google Drive 文件夹同步过去,装好翼德跑一次 onboard,他会“认出你”,不用重配。
- **不想自动放歌?** 直接跟翼德说“静音奖励”。
- **整理记忆要我管吗?** 不用,翼德每天第一次开工自动整理。
- **卡住了?** 直接在对话框问翼德,或找咕鸡。
